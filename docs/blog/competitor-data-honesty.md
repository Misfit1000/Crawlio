# Competitor data honesty

Competitor pages provide content-gap observations only. `High traffic` is allowed only with a named approved data source and observation date. Otherwise the platform uses labels such as `Authoritative reference`, `Widely covered`, or `Traffic data unavailable`.

The workflow does not infer traffic, ranking probability, clicks, or search volume, and it may not copy wording, structure, examples, tables, diagrams, or an author's style.

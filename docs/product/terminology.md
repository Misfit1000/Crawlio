# Product Terminology

| Use | Meaning |
| --- | --- |
| Audit | One submitted website review |
| Finding | A result identified by an audit check |
| Recommendation | A suggested correction for a finding |
| Affected pages | URLs associated with a finding |
| Audit service | Customer-facing name for background processing |
| Completed with warnings | Results are available but some evidence was unavailable |
| Metadata preview | A composition built from public page metadata |
| Passive security | Non-invasive review of public browser and response signals |

Do not call metadata previews screenshots, response timing Core Web Vitals, or passive checks penetration tests.

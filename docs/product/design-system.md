# Product Design System

## Structure

Use one H1, one primary action near the page title, and supporting actions with lower visual weight. Summaries may use cards; repeated findings use rows or tables; detailed evidence uses drawers or plain sections.

## Visual rules

- Controls use 8px corners; standard panels use 10-12px corners.
- Shadows remain subtle and secondary to borders and surface contrast.
- Blue identifies primary actions. Emerald, amber, and red are reserved for status.
- Page spacing follows 4, 8, 12, 16, 24, 32, 48, and 64px increments.
- Animations communicate state changes and respect reduced-motion preferences.
- Mobile layouts stack controls, preserve 44px touch targets, and never rely on horizontal page scrolling.
